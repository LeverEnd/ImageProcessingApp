# ImageProcessingApp

A complex, cloud-native application built for my thesis. It uses an automated backend configured with Infrastructure as Code (Terraform), paired with a dynamic web frontend.

## Project Overview

I developed a complex backend environment for an image processing web application using AWS web services, accompanied by a simple web frontend. Building such an environment manually is prone to errors and can be highly time-consuming. To avoid this, I implemented the backend using Infrastructure as Code (IaC) solutions to ensure automation. I achieved this with two separate tools, Terraform and CloudFormation. First, I designed and built the environment using Terraform, and then I converted the finalized code into CloudFormation. The backend itself includes object storage, a NoSQL database, serverless computing components, a state machine, a monitoring service, a notification system, and an API gateway.

After completing the application, I used two generative artificial intelligence models, DeepSeek and Gemini, to evaluate the extent to which they can assist human developers in creating an IaC environment. Initially, I prompted them to generate the Terraform environment based on a detailed description, corrected their mistakes using further prompts, and then asked them to translate the finalized code into CloudFormation, where I performed debugging once again. The evaluation was based on a custom scoring system, analyzing factors such as the time spent creating the environment, code correctness, and context retention. Finally, I analyzed the results and drew conclusions. Ultimately, I concluded that generative AIs significantly accelerate development; however, substantial differences were observed between the models regarding API handling and automation capabilities.

## Repository Structure

* **Terraform_Backend** & **CloudFormation_Backend**: My handwritten, baseline IaC code.
* **Frontend**: My handwritten web application for the user interface.
* **Gemini_Generated_Code** & **DeepSeek_Generated_Code**: Experimental data. These folders contain the raw IaC outputs generated by the AIs during the testing phase of my thesis for comparative analysis.
